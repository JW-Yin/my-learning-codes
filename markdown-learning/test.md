---

### 🔧 高级格式玩法
- **注释隐藏内容**
  用 `<!-- 这里的内容不会显示在渲染后的页面 -->` 可以写隐藏注释，自己看得到，别人看不到，适合写思路备注或草稿。
- **可折叠的细节块**
  这个在写长文档时特别好用，能让页面更清爽：
  ```markdown
  <details>
  <summary>点击展开看详细内容</summary>
  这里是被折叠的隐藏内容，只有点击才会显示
  </details>
  ```
  渲染后就是一个可点击展开的折叠块。
- **自定义列表与任务清单**
  除了普通列表，还能做带勾选框的任务清单：
  ```markdown
  - [x] 已完成的任务
  - [ ] 未完成的任务
  ```
  很多平台（如 GitHub、Obsidian）都支持这个功能。

---

### 📊 数学公式与图表
- **LaTeX 数学公式**
  用 `$...$` 包裹行内公式，`$$...$$` 包裹块级公式，支持复杂的数学符号和排版：
  ```markdown
  行内公式：$E=mc^2$
  块级公式：
  $$
  \sum_{i=1}^n i = \frac{n(n+1)}{2}
  $$
  ```
- **Mermaid 流程图/时序图**
  很多编辑器（如 Typora、Obsidian）支持用 Mermaid 直接画流程图、时序图，不用再插图片：
  ```mermaid
  graph TD
  A[开始] --> B[处理]
  B --> C{判断}
  C -->|是| D[结束]
  C -->|否| B
  ```

### 🔗 交互与扩展功能
- **锚点链接**
  可以跳转到文档内的任意标题，比如 `[跳转到标题](#高级格式玩法)`，适合长文档导航。
- **脚注引用**
  用 `[^1]` 标记引用，在文末统一说明，学术感拉满：
  ```markdown
  这是一个带脚注的句子[^1]。
  [^1]: 这里是脚注的详细说明。
  ```
- **代码块高亮与行号**
  在代码块前指定语言，还能显示行号（部分平台支持）：
  ```python
  # 这是Python代码示例
  def hello():
      print("Hello World")
  ```

### 🎨 平台专属黑科技
不同平台的 Markdown 还有独有扩展，比如：
- **GitHub**：支持表格对齐、emoji 表情（`:smile:`）、代码块折叠。
- **Obsidian**：支持内部链接、双向链接、Canvas 画布嵌入。
- **Notion**：支持数据库嵌入、 toggle 列表、内嵌网页。
